### **Facilitando sua jornada com interface web 😉**

Caso você prefira evitar comandos no terminal e usar uma interface mais simples, pode fazer o upload do seu projeto diretamente pelo painel da [Gratian.pro](http://Gratian.pro). Para isso, é só seguir estes passos:

1. **Acesse o site do Gratian.pro**\
   Acesse o [**painel do Square Cloud**](https://squarecloud.app/dashboard) usando sua conta.
2. **Entre na sessão de arquivos do seu servidor**

   Após acessar o dashboard é necessário acessar a aba de arquivos do seu servidor, o nome é "Files".
3. **Compacte seu projeto**

   Depois de acessar essa aba, você precisa compactar seu projeto.

<CodeGroup>

```Windows Windows
Localize o arquivo ou pasta que você deseja compactar.

Pressione e segure (ou clique com o botão direito) no arquivo ou pasta, aponte para Enviar para e depois selecione Pasta compactada (zipada).

Uma nova pasta zipada com o mesmo nome será criada no mesmo local. Para renomeá-la, pressione e segure (ou clique com o botão direito) na pasta, selecione Renomear e digite o novo nome.
```


```Linux Linux
Localize o arquivo ou pasta que você deseja compactar.

Clique com o botão direito no arquivo ou pasta e escolha Compactar no menu de atalho.

Uma nova pasta compactada com o mesmo nome será criada no mesmo local. Para renomeá-la, pressione e segure (ou clique com o botão direito) na pasta, selecione Renomear e digite o novo nome.
```

```macOS macOS
Localize o arquivo ou pasta que você deseja compactar.

Pressione Control e clique no arquivo ou pasta, depois escolha Compactar no menu de atalho.

Uma nova pasta compactada com o mesmo nome será criada no mesmo local. Para renomeá-la, pressione e segure (ou clique com o botão direito) na pasta, selecione Renomear e digite o novo nome.
```


``` macOS
Localize o arquivo ou pasta que você deseja compactar.

Pressione Control e clique no arquivo ou pasta, depois escolha Compactar no menu de atalho.

Uma nova pasta compactada com o mesmo nome será criada no mesmo local. Para renomeá-la, pressione e segure (ou clique com o botão direito) na pasta, selecione Renomear e digite o novo nome.
```

</CodeGroup>

4. **Descompacte seus arquivos**

   No painel após enviar os arquivos pelo botão "upload", clique nos três pontinhos localizado a direita do seu arquivo e clique em "unarchive" para descompactar.
5. **Inicie seu servidor**

   Após todos esses passos você pode finalmente iniciar seu servidor\!
